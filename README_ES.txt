TRACKER ENTRENAMIENTO + MENÚ (GRATIS)

Qué es:
- Un "programa" que funciona en cualquier navegador (PC y móvil).
- No necesitas instalar nada ni crear cuentas.
- Se guarda en el dispositivo (localStorage).
- Puedes exportar backup y registros.

Cómo usarlo (modo fácil):
1) En PC:
   - Abre index.html con doble clic.
2) En móvil:
   - Opción recomendada: súbelo a GitHub Pages (gratis) y ábrelo como una web.
   - Si quieres, añade a pantalla de inicio.

GitHub Pages (pasos simples):
1. Crea cuenta en GitHub (si no tienes).
2. Crea un repo nuevo (ej: tracker-entreno-menu).
3. Sube estos archivos (arrastrar y soltar).
4. Settings → Pages → Source: Deploy from a branch → Branch: main / root → Save.
5. Te dará una URL. Ábrela en el móvil.

Uso semanal:
- Pega el WhatsApp en la caja y pulsa “Importar semana”.
- Ajusta el “Tipo de día” si hace falta.
- Marca checks (Entreno hecho, Comidas hechas).
- En Registrar, añade series/pesos.
- En Resumen, copia y envía al entrenador.

Macros:
- Si quieres que calcule kcal/prote/carb/grasa, rellena esos números en el menú base.
  (En esta app vienen vacíos por defecto.)
